# Problem 247 - Tenth Line

Given a text file `file.txt`, print only the 10th line.

## Example

Assume `file.txt` has the following content:

```text
Line 1
Line 2
Line 3
Line 4
Line 5
Line 6
Line 7
Line 8
Line 9
Line 10
```

Expected output:

```text
Line 10
```

## Notes

1. If `file.txt` contains fewer than 10 lines, output nothing.
2. Try to solve it in at least 3 different ways.
